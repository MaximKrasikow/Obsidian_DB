# Общие сведения
`venv` — это **встроенный модуль Python 3.3+**, создающий изолированные окружения для проектов.
Каждое окружение — это **копия интерпретатора Python**, в которой есть собственный каталог `site-packages`, куда устанавливаются зависимости (библиотеки).


> [!_] Синтаксис
> ``` python
><<СОЗДАНИЕ>>
> 	python<ВЕРСИЯ ИНТЕРПРЕТАТОРА> -m  venv [ИМЯ ОКРУЖЕНИЯ]
> 
><<АКТИВАЦИЯ>>
>	[ИМЯ ОКРУЖЕНИЯ]\Scripts\activate
>
><<ДЕКАТИВАЦИЯ>>
>	deactivate
>```
