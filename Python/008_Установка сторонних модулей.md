# Общие сведения
Сайт со всеми официальными пакетами [https://pypi.org](https://pypi.org)
Для скачивания сторонних модулей используется команда `pip install [НАЗВАНИЕ ПАКЕТА] <== ВЕРСИЯ> `

	1. Если не указывать версию скачивается самая последняя


# Просмотр установленных модулей
`pip list`

# Установка пакетов из названий в текстовом файле
`pip install -r <текстовый файл>`


# Отображений названий и версий пакетов текущего окружения
`pip freeze > [ИМЯ ФАЙЛА].txt`
