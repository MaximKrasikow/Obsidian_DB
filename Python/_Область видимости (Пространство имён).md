# Встроенная
Встроенная (**built-in**) область видимости отвечает за переменные и функции на уровне интерпретатора. В частности, в любом Python-проекте мы всегда можем обратиться к True, print, map и т.д. Эти объекты мы лично не объявляли, ниоткуда не импортировали, но спокойно используем.
``` python
file.py

Мы нигде не объявляли эти имена но можем ими пользоваться
print()
type()
...
и .т.д
```
