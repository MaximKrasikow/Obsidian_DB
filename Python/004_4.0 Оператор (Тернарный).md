# Общие сведения
==Всегда== Возвращает значение по условию

	1. Можно использовать в других конструкциях языка (Конструкторы, Генераторы...)

> [!Note] Синтаксис
> 	[Выражение] if [Условие] else [Выражение]
> Выражение — может быть также тернарным
> Является сокращением от:
> ``` python
> if A > B:
> 	eXm = A
>else:
>	eXm = B
>```
