# Аргумент для символа разделителя в строке `sep`

sep (separator — «разделитель»)
```
print("Добрый день, ", name, ".", sep="")

Значение по умолчанию для него — символ "пробел"
Примечание в аргументах не ставятся пробелы между аргументами и знаком =
```
