Переменная — Ссылка на объект?

В языке Python переменные не хранят значения, а лишь ссылаются на них!

# Правильные имена переменных
Имена переменных являются регистр зависимыми!

1. Имена следует брать существительными (отвечают на вопросы: кто, что): apple, count, weight
2. Имена должны быть осмысленными и отражать суть данных.
3. Допустимые символы в именах: первый символ – любая буква латинского алфавита a-z, A-Z и символ подчеркивания _. В качестве второго и последующих символов еще цифры 0-9:
   Правильно:
   count_1, count_index_1, count123
   Неправильно:
   !count, 



# Создание переменной
	1. Для создания переменной используется оператор "=" который связывается 2? операнда между собой.
	2. Операция создания переменной является правоассициативной.

Примеры:

1. a = 1 ->
2. a = 2.5 # Тип переменной вычисляется автоматически 
3. b = a #  Для переменной b не создаётся новый объект 2.5, а лишь создаётся новая ссылка на уже созданный объект.



# Каскадное присваивание
a=b=c=2.5


# Множественное присваивание
a, b = 1, 2.5
