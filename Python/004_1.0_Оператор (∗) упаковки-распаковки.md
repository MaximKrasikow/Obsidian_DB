# Распаковка
С помощью `*` мы можем РАСПОКОВАТЬ ???любой итерируемый объект


``` python
list = [1, 2, 3]
dict = {"A":1, "B":2}
tuple = (1, 2, 3)

*list --> 1 2 3
*dict --> 'A' 'B'
*tuple --> 1 2 3

emx1 = (*list,) --> (1, 2, 3)
```

# Упаковка
С помощью \* мы можем упаковывать несколько значений в ???СПИСОК


``` python
x, *y = (1, 2, 3, 4, 5) --> x=1 y=[2, 3, 4, 5]
*x, y = (1, 2, 3, 4, 5) --> x=[1, 2, 3, 4] y=5

x, *y = "ABC" --> x="A" y =["B", "C"]
```
