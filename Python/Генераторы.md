# Общие сведения

	1. Генератор является итерируемым объектом
	2. По генератору можно пройтись только 1 раз, после чего он исчезает из памяти
	3. Элементы генератора создаются по ходу обращения к ним а не сразу в момент создания генератора.

# Функция-генератор
**Функция-генератор** — это специальный тип функции в Python, которая возвращает **объект-генератор** вместо немедленного выполнения и возврата результата.

## Оператор (yield)

	1. При наличии в функции yield вызов функции **не** выполняет её код — возвращается generator object.
	2. 

```python
def simple():
    print("start")
    yield 1
    print("after 1")
    yield 2
    print("before return")
    return 3

A = simple()
A --> <generator object simple at 0x00000281CBB8C880>

A = next(simple) --> start
print(A) --> 1
```
