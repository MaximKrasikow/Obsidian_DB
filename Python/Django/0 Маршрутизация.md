# Общие сведения


# URL
Проверяются в порядке следования

2-я проверка не выполнится никогда так как под 1 подходят все комбинации url
```
urlpatterns = [
>> Проверится 1
    path('test/<slug:test_slug>/', views.index),
>> Проверится 2
	path('test/<int:test_int>/', views.index)
]
```

## Конверторы
необходимы для создания динамических url
`path('test/<int:test_int>/', views.index)`
