# Литерал

==Литералы== - 
значения, представленные в коде программы, например, числа или строки
```
5         # Целочисленный литерал
3.4       # Литерал в виде числа с плавающей точкой
'строка'  # Строковый литерал
```

# Оператор

==Оператор== -
Элемент выполняющий дейтсвия над оперндами

```
=
+
-
```

# Операнд

То над чем выполняются действия

```
1 + 2

1 и 2 операнды
```
