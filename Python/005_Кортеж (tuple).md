# Общие сведения
Кортежи (tuple) (Последовательность | Структура данных | Коллекция) – это неизменяемые последовательности
Кортеж храни все себе не сами объекты а ссылки на эти объекты в памяти.
Кортеж является оптимизированным и работает быстрее списка


	1. Неизменяем НО возможно изменение элементов (изменяемого типа);
	2. Элементы могут дублироваться;
	3. Хранит в себе любой??? тип данных;
	4. Вложенность элементов любая???
	5. Упорядочен
	6. Итерируемый
