# Параметры-Аргументы функции
## Общие сведения
==Параметры== — записываются при объявлении функции.
==Аргумент==ы — передаются в функцию при её вызове.
==Формальный параметр== — параметр со значением по умолчанию.
==Фактический параметр== — остальные параметры.


	1. ф

> [!NOTE] Порядок следования параметров
> ``` python
> def func(<ПОЗИЦИОННЫЕ>, <*args>, <ИМЕНОВАННЫЕ>, <**kwargs>):
> 	pass
> <ПАРАМЕТРЫ ПО УМОЛЧАНИЮ> — могут находиться в любом месте
> 
> В Python 3.8+ можно ограничить способ передачи аргументов:
> def func(<ТОЛЬКО ПОЗИЦИОННЫЕ> /, <ЛЮБЫЕ> *, <ТОЛЬКО ИМЕНОВАННЫЕ>):
> 		pass
> ```

> [!NOTE] Правила задания аргументов при вызове функции
> ``` python
> [ИМЯ ФУНКЦИИ] (<ПОЗИЦИОННЫЕ>, <ИМЕНОВАННЫЕ>)
> ```

## Типы параметров

### Позиционные параметры (Фактические параметры)

``` python
def exm_def (exm1, exm2): 
	print(exm1, exm2)

exm_def("1", "2") --> 1 2
```

> [!Warning] Ошибка
> ``` python
> def exm_def (exm1, exm2):
> 	print(exm1, exm2)
> 
> exm_def("1") --> Вызовет ошибку
> ```
> 
### Параметры по умолчанию (Формальные параметры, Ключевые)

Можно указать параметру значение по умолчанию

``` python
def exm_def (exm1, exm2="B"): 
	print(exm1, exm2)

exm_def("A") --> A B
exm_def("A", "ABC") --> A ABC
```

### Произвольное число параметров
#### \*args
Позволяет принимать любое количество ПОЗИЦИОННЫХ параметров упаковывая их в КОРТЕЖ.

	1. Имя `args` — это соглашение, можно использовать любое другое (например, `*numbers`).

``` python
def exm_def (*args): 
	print(args)

exm_def("1", "2") --> (1, 2)
```

#### \*\*kwargs
Позволяет принимать любое количество ИМЕНОВАННЫХ параметров упаковывая их в СЛОВАРЬ.

``` python
def exm_def (**kwargs): 
	print(kwargs)

exm_def(A="1", B="2") --> {'A':'1', 'B':'2'}
```
