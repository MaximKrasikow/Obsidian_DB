# Общие сведения
Пакет (package) — специальным образом организованный каталог файлов
Пакет объявляется располагающимся в нём инициализирующим файлом `__init__.py`

	1. При импорте пакета мы получаем все переменные находящиеся в файле `__init__.py`


``` python
С:\Test\PACKAGE\__init__.py

	import PACKAGE.exm_pack_1
	NAME = "Это пакет 1"

-----
С:\Test\PACKAGE\exm_pack_1.py
	
	def get_pack():
		print("Это часть пакета 1")

-----
С:\Test\exm.py

	import PACKAGE
	PACKAGE.NAME --> Это пакет 1
	PACKAGE.exm_pack_1.get_pack() --> Это часть пакета 1
	
```
