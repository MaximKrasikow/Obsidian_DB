# Операции со списками
## Создание списков
Чтобы создать список, нужно перечислить его элементы через запятую в квадратных скобках:

``` python
1. eXm = [] | list() --> 1 способ работает в 2 РАЗА быстрее!
2. eXm = [0, 1, 2, 'a', (1, 2, 3)]
3. 
```

### Генераторы списков (Списочные выражения | List comprehensions)

	1. Работате быстрее for и while

``` python
[ [Выр. | Объект] for [Пер.] in [Итерирумый Объект] 
<if> [Усл.] <and | or> [Усл.] ] 

	Выражение — Может быть также тернарным оператором

>> Пример
[i * 2 for i in range(0, 5) if i != 0] --> [2, 4, 6, 8]
```

## Обращение к элементам списков
При работе со строками мы использовали **индексацию**, то есть обращение к конкретному символу строки по его индексу. Аналогично, можно индексировать и списки.

``` python
example = [0, 1, 2, 3, …] — прямая последовательность
example = [-4, -3, -2, -1 …] — обратная последовательность
```

## Сложение списков
``` py
[1, 2, 3] + [4, 5, 'A'] --> [1, 2, 3, 4, 5, 'A']
```

## Умножение списка
``` py
eXm = [1, 2]
A = eXm * 2 --> [1, 2, 1, 2]
```

## Изменение элементов списка

Изменение возможно с помощью [[00 Срезы]]
``` python
eXm = [0, 1, 2, 3]

eXm[:2] = ['A', 'B'] --> eXm = ['A', 'B', 2, 3]
```
## Удаление элементов
С помощью оператора `del` можно удалять элементы списка по определенному индексу.
example = [1, ...]
del example[0] / del example[x:y:z]

## Вывод списков

В Python есть удобный способ вывода элементов списка без использования цикла `for`.
numbers = [0, 1, 2, 3, ]
print(*numbers) # 0 1 2 3
