# Методы словарей
## Метод (.fromkeys)
Назначает нескольким ключам одинаковое значение.

	1. Вовращает словарь
	2. Не изменяет словарь а создаёт новый из класса dict

> [!Синтаксис] 
> dict.fromkeys([ИТЕРИРУЕМЫЙ ОБЪЕКТ], <None or ЗНАЧЕНИЕ>)
> ``` python
> 1. exm = dict.fromkeys(["1", 2], "ABC") --> {"1": "ABC", 2: "ABC"}
> ```

## Метод (.value)
Возвращает объект dict_value.

> [!NOTE]
> [СЛОВАРЬ].keys()
> ``` python
> 1. exm = {"a":"b", 2: 3}
> 2. exm.value() --> ["b", 3]
> ```

## Метод (.items)
Возвращает объект dict_items

> [!NOTE] Синтаксис
> [СЛОВАРЬ].items()
> ``` python
> 1. exm = {"a":"b", 2: 3}
> 2. exm.items() --> [("a", "b"), (2, 3)]
> ```

## Метод (.get)

### Для словарей
Получение значения из словаря по ключу.

> [!NOTE]
> [СЛОВАРЬ].get([КЛЮЧ], <None or ОШИБКА_ВОЗВРАТ>)

## Метод (.setdefault)

### Для словарей
Получает значение из словаря или добавляет при отсутствии.

> [!NOTE]
> [СЛОВАРЬ].setdefault([КЛЮЧ], <None or ЗНАЧЕНИЕ ПО УМОЛЧАНИЮ>) 
>
> - По умолчанию добавляет None
> 
> ``` python
> 1. exm_1 = {"a":"b"} 
> 2. exm_1.setdefault("a") --> b
> 3. exm_1.setdefault("b", "ABC") --> exm_1 == {"a":"b", "b":"ABC"} 
> ```

## Метод (.update)


### Для словарей
Пересечение двух словарей в 1.
`СЛОВАРЬ_ОБЪЕДИНЕНИЯ.update(СЛОВАРЬ)` 
> 1. Изменяет словарь слева.

``` python
exm_1 = {"a":"b"} exm_2 = {"a":"ABC", "c":"d"}
exm_1.update(exm_2) --> exm_1 == {"a":"ABC", "c":"d"}
exm_1 |= exm_2 --> АНАЛОГИЧНО 
```


## Метод (.pop)
Возвращает элемент коллекции и ==удаляет== его

### Для списков

### Для словарей

``` python

```

## Метод (.popitem)
### Для словарей
Получает ==**последний добавленный элемент==** ключ и значение в виде кортежа.
`СЛОВАРЬ.popitem()` 

	1. Пустой словарь вызывает ошибку.

``` python
exm_1 = {"a":"b", "C":"ABC"} 
exm_1.popitem() --> ("C", "ABC") & exm_1 == {"a":"b"}
```
