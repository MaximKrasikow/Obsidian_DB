
# Функция (enumerate)
## Общие сведения
Позволяет одновременно получать **индекс** и **значение** элемента при итерации по итерируемому объекту (списку, кортежу, строке и т.д.). 
Это избавляет от необходимости вручную отслеживать счётчик.

	1. 

``` python
[ИНДЕКС],  [ЗНАЧЕНИЕ] = enumerate([ОБЪЕКТ], <start=[НАЧ. ИНДЕКС]>) 
--> Возвращает объект итератор

ОБЪЕКТ — это итерируемый объект, который нужно перебрать.
НАЧ. ИНДЕКС — По умолчанию 0

Если задать 1 значение слева вернёт кортеж ([ИНДЕКС], [ЗНАЧЕНИЕ]) 
```

ПРИМЕР:
``` python
1. exm = [1, 2, 'ABC']
2. for index, value in enumerate(exm):
	1. print(index, value)
--> 0 1
	1 2
	2 ABC

3. tpl = enumerate(["ABC"]) --> tpl == (0, 'ABC')
```
``` python
[ИНДЕКС],  [ЗНАЧЕНИЕ] = enumerate
```
