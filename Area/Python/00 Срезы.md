# Общие сведения
В общем случае срез представляет собой
ОБЪЕКТ\[<ЛЕВЫЙ ИНДЕКС> : <ПРАВЫЙ ИНДЕКС> : <ШАГ СРЕЗА>]

	1. Левый индекс включительно
	2. Правый индекс не включительно
	3. Не возвращает ошибки при выходе за диапазон
	4. Стандартно индекс  увеличивается СЛЕВА --> НАПРАВО


``` python
>> Для любого базовго типа???
         -5 -4 -3 -2 -1	
1. exm = "A  B  C  D  E"
          0  1  2  3  4

2. exm[0:] & exm[::] --> "ABCDE"
3. exm[0::2] --> С 1 эл. через +2 "ACE" Работает с отрицательными индексами.
                    
4. exm[0::-1] --> "EDCBA"
5. exm[0:-3] -->  ДО -3 "AB"
6. exm[-3:] -->  С -3 "CDE"
   
7. exm[0:2:-1] --> ДО 2 индекса НЕ вкл. так как правый индекс "ED"
8. exm[2::-1] --> СО 2 индекса вкл. так как левый индекс "CBA"
   
9. exm[-3:-1] --> "CD"
10. exm[-2:-4:-1] --> "DC"
   

```

# Работа со срезами
## Копирование срезом

### Копирование срезом для (Списков) 

``` python
1. eXm [:] & eXm [::] & eXm [0:]
   eXm [x:] & eXm [:x] & eXm [::x] & eXm[::-1] --> Создаст новый объект списка
```

### Копирование срезом для (Строк) 

``` python
1. eXm [:] & eXm [::] & eXm [0:] --> Вернёт ссылку на строку
eXm [x:] & eXm [:x] & eXm [::x] & eXm[::-1] --> Создаст новый объект
```
