## Работа со строками

### Конкатенация (сложение)
	1. Складывает 2 и более строки с помощью оператора +.

```
s1 = "a"
s2 = "b"

s3 = s1 + s2 # ab
```

### Умножение
	1. Допустимо только умножение на int.


```
s1 = "a"
s2 = "b"

s3 = s1 * 5  # aaaaa
```
