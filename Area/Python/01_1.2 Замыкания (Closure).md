# Общие сведения


Для использования замыканий важно соблюдение **условий**:  

- у нас должна быть дочерняя функция (внутри другой функции),
- дочерняя функция должна обращаться к переменной, объявленной в вышестоящей функции,
- функция-родитель возвращает наследуемую функцию.



``` python
>> Функция-обёртка
def closure_example():
    x = 11
    def inner():
        print(f'Переменная из замыкания: {x}')
    return inner

closure_example()() --> Переменная из замыкания: 11

-----

def closure_example():
    x = 11
    def inner():
        print(f'Переменная из замыкания: {x}')
    return inner

x = closure_example()
x() --> Переменная из замыкания: 11
```
