# Общие сведения

>**Встроенные (служебные) атрибуты** - методы и свойства унаследованные от общего для всех классов в Python родительского класса `object`. Многие из этих атрибутов можно переопределить внутри своего класса.

> 1. Атрибуты (поля и методы), имена которых обрамляются `__`, Python трактует как специальные.
> 2. Такие методы выполняются автоматически при создании объекта класса


# Методы
# Метод инициализатор (\_\_init__)
Вызывается в начале  создания экземпляра
Позволяет нам сразу без дополнительного создаваемого метода, заполнять экземпляр начальными данными.

>[!_] Синтаксис
>	__init__([self], <ПАРАМЕТРЫ>)
>``` python
>class Tank:
>	def __init__(self, name):
>		self.name_tank = name
>
>t_34_obj = Tank("T-34")
>print(t_34_obj.name_tank) --> T-34
>```

# Метод финализатор (?деструктор) (\_\_del__)


>[!_] Синтаксис
>	__init__([self], <ПАРАМЕТРЫ>)
>``` python
>class Tank:
>	def __del__(self):
>			print(f"Был удалён объект {self} с именем ")
>
>t_34_obj = Tank("T-34")
>print(t_34_obj.name_tank) --> T-34
