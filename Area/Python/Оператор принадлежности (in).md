# Общие сведения
Проверяет принадлежность элемента к последовательности или коллекции и возвращает булево значение True или False.

``` python
[ОБЪЕКТ] in [ИТЕРИРУЕМЫЙ ОБЪЕКТ]
``` 

	1. Является регистрозависимым

# Для типов данных
## Для списков
Позволяет проверить вхождение элемента в список


``` python
1. eXm = [1, 2, 3 ['A', 2]]

2. 1 in eXm --> True
3. ['A', 2] in eXm --> True
4. 5 in eXm --> False


```

# Для строк
Позволяет проверить вхождение символа? в строку.


``` python
Проверяет полно вхождение подстроки в строку
'ab' in "abracadabra"  # True
'abc' in "abbc"        # False
 ```

# Для словарей
Оператор принадлежности `in` на словарях работает **очень быстро**, намного быстрее, чем на списках

``` python
Проверяет наличие КЛЮЧА
 1 in {1:'2'} --> TRUE
'abc' in {'abc':"abbc"} --> TRUE
 ```
