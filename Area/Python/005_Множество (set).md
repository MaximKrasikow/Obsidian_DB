# Основные данные
Множество (Коллекция) — структура данных, организованная так же, как математические множества.

	1. Изменяемый
	2. Все элементы различны (уникальны)
	3. Все элементы должны быть неизменяемыми типами (ХЕШИРУЕМЫМИ)
	4. Множество неупорядоченно ==НЕ ИМЕЕТ ИНДЕКСОВ==
	5. Операция конкатенации `+` и умножения на число `*` ==недоступны== для множеств.
