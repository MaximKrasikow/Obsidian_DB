# Атрибуты (Свойства | Поля) класса
Поля можно условно разделить на две группы:

## Статические
>**Статические** - поля класса, которые объявляются внутри тела класса и создаются тогда, когда создается класс.

## Динамические
>**Динамические** - поля экземпляра. Для создания динамического поля необходимо обратиться к `self` внутри метода.



## Операции с атрибутами

## Обращение к атрибуту

	 1. Нельзя обратиться к несуществующему атрибуту

``` python
def Tank:
	name = "M1"	

> Обращение к атрибуту
EXM = Tank.name
print(EXM) --> M1

> Чтобы избежать исключения
getattr([ИМЯ Класса], [ИМЯ Атр.], [Знач. по умол.])

> Атрибуты изменяемы
Tank.name = "T-34"

> Функция проверки наличия
hasattr([Класс], [Атр.]) --> True | False

```

## Создание атрибутов

``` python
def Tank:
	name = "M1"	

> Добавит новый атрибут или перезапишет значение
Tank.fuel = 100

setattr([ИМЯ КЛАСССА], [ИМЯ Атр.], [ЗНАЧЕНИЕ])

```

## Удаление атрибутов

``` python
def Tank:
	name = "M1"	

> Нельзя удалить несуществующий
del Tank.name

```

## Особенности атрибутов

	Каждый экземпляр создаёт своё пространство имён

![[Ссылочная модель атрибутов]]
Изначадбгл экземпляры не содержат собственных атрибутов а лишь ссылаются на атрибуты родитьельского класса (?принцип действия)
(?При обращении к атрибутам класса идёт иерархическое обращение)


> [!_] Доп. инф.
> ```python
> <<Статический и динамический атрибут может иметь одно и то же имя:
> class Tank:
>     # статический атрибут
>     name = "Default name"
> 
>     def __init__(self, name):
>         # динамический атрибут
>         self.name = name
> 
> 
> t_5 = Tank("M5")
> 
> print(Tank.name) --> Default name
> print(t_5.name) --> M5
> 
> <<Статический атрибут можно переопределить не создавая объект класса:>>
> class Tank:
>     name = "Default name"
> 
> Tank.name = "None"
> print(Tank.name) --> None
> ```
