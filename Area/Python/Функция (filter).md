# Общие сведения
Возвращает только те элементы которые интерпретируются как True функцией


>[!Note] Синтаксис
>1. Возвращает итератор
>
>```python
>filter( [ССЫЛКА НА ФУНКЦИЮ | МЕТОД], [ИТЕРИРУЕМЫЙ ОБЪЕКТ] )
>
>[ФУНКЦИЯ] -- ВОВЗРАЩАЕТ TRUE или FALSE
>[МЕТОД] — можно указат метод вовращающий булево значение (напр. str.isalpha)
>```
