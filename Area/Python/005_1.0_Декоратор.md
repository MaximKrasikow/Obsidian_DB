# Общие сведения
>Декоратор - это **функция**, принимающая на вход **функцию**, и возвращающая (другую) **функцию**

Декораторы позволяют вам выполнить произвольный код до/после/вместо вызова функции, модифицируя ее входные аргументы, результат выполнения и добавляя различные сайд-эффекты


	1.??? Все декаораторы являются замыканиями, НО не все замыкания являются декораторами.


> [!NOTE] Синтаксис
> ``` python
> 	def simple_decorator(func):
> 		def inner():
> 			print('Начало работы декоратора...')
> 			A = func()
> 			print('Декоратор отработал!')
> 			return A
> 		return inner
> 	
> 	def print_hi():
> 		return 'Привет, я - функция, которую задекорировали!'
> 	    
> <<При декорировании мы теряем имя оригинальной функции `print_hi`>>
> 	print_hi = simple_decorator(print_hi)
> 	print_hi() 
> 	--->
> 	Начало работы декоратора... 
> 	Привет, я - функция, которую задекорировали! 
> 	Декоратор отработал!
> 	
> -----
> 	
><<Можно использовать ?оператор @ для декорации функции>>
> 	@simple_decorator
> 	def print_hi():
> 		print('Привет, я - функция, которую задекорировали!')
> 	
> 	print_hi()
> 	Вывод аналогичен
> ```


# Декорирование функции с параметрами

``` python
def simple_decorator(func):
    def inner(*args, **kwargs):
        print(f'Декорируем функцию с параметрами: {args}, {kwargs}')
        func(*args, **kwargs)
        print('Все прошло успешно!')
    return inner
 
@simple_decorator
def adder(*nums):
    print(sum(nums))
```

# Декоратор с параметрами

``` python
def repeater(num_of_repeats=1):

    def outer_decorator(func):

        def inner_decorator(*args, **kwargs):

            if num_of_repeats > 0:

                for _ in range(num_of_repeats):

                    func(*args, **kwargs)

            else:

                func(*args, **kwargs)

        return inner_decorator

    return outer_decorator

  
  

@repeater(3)
def print_text(message):
    print(f'Вам сообщение: {message}')

print_text('Просыпайся!')
--->
Вам сообщение: Просыпайся!
Вам сообщение: Просыпайся!
Вам сообщение: Просыпайся!
```
