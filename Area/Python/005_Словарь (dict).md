# Основные данные
Словарь ("Ассоциативный массив",  "Хеш таблица") — 

	1. Изменяемый;
	2. Индексация по ключу;
	3. Ключи только неизменяемый (хешируемый???) объект;
	4. Значение любой тип данных, любой вложенности;
	5. Операция конкатенации `+` и умножения на число `*` недоступны для словаре;
	6. Элементы упорядочены

# Операции со словарями
## Создание
``` python
1. exm = {'A': 'ABC', 1: 2}
2. exm = {} <-> dict()
3. exm = dict(<ИМЯ_КЛЮЧА = ЗНАЧЕНИЕ>, ...)
4. exm = dict( [ ('KEY', 'VALUE'), ... ] )
5. exm = dict( ( ['KEY', 'VALUE'], ... ) )

```

## Обращение к элементам
``` python
1. exm = {'Python': 'Гвидо ван Россум', 1: 2}
2. (По индексу/ключу) example['Python'] --> Гвидо ван Россум
```

## Добавление элементов
``` python
1. exm = {}
2. (По ключу) exam[1] = 'A' --> exm == {1:"A"}
```
