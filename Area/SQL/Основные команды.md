# INSERT INTO

``` SQL
INSERT <INTO> [ТАБЛИЦА] < ( АТРИБУТ, .....) >
VALUES 
( [ЗНАЧЕНИЕ_АТР.], ..... ),
...
(...)
```

# SELECT

``` SQL
SELECT [* | АТРИБУТЫ, | ] FROM [ТАБЛИЦА]

> Модификатор названия атрибута итоговой таблицы  AS
SELECT [АТРИБУТ] AS [НОВОЕ НАЗВАНИЕ], ... FROM [ТАБЛИЦА]

> В результ. таблицу можно добавить вычисляемый атрибут
SELECT [АТРИБУТ] [ОПЕРАТОР (* | + | -)] [АТРИБУТ] <AS [ИМЯ]>, ... FROM [ТАБЛИЦА]
```

# DISTINCT
Выбирает уникальные значения атрибута

``` SQL
SELECT <DISTINCT> [АТР.]
FROM [ТАБ.]
```
